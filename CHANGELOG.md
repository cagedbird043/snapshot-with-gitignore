# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-11

### Added

- Initial public release of Snapshot With Gitignore
- Smart filtering with `.gitignore` support (including nested gitignore files)
- Drag and drop interface for easy folder selection
- Project tree visualization with file sizes
- Web Worker-based background processing
- One-click copy to clipboard functionality
- Download snapshot as Markdown file
- Real-time file preview before generation
- Customizable ignore rules with live reload
- Built-in ignore patterns for common directories and files
- Support for multiple `.gitignore` files in nested directories
- Comprehensive README documentation (English & Chinese)
- MIT License
- Contributing guidelines
- Complete project metadata and package information

### Features

- **Smart Filtering**: Automatically respects all `.gitignore` rules
- **Performance**: Uses Web Workers for non-blocking file processing
- **User-Friendly**: Drag-and-drop support for quick folder selection
- **Customizable**: Edit ignore rules on the fly
- **AI-Optimized**: Generates output perfect for AI conversations
- **Cross-Platform**: Works on Windows, macOS, and Linux

### Technical Stack

- React 19.0
- TypeScript 5.8
- Vite 6.2
- Web Workers API
- Native CSS with CSS Variables
